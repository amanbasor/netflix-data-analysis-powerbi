Netflix Data Analysis Dashboard (2021)                
Project Overview     

This project analyzes Netflix’s 2021 dataset to uncover insights about content trends, audience segmentation, and global distribution. The data was cleaned using Python and visualized using Power BI.

Objectives      
Compare Movies vs TV Shows
Analyze trends over time
Identify top content-producing countries
Understand audience categories

Tools Used        
Power BI Desktop
Python (Google Colab)
DAX
CSV Dataset (Kaggle)

Dashboard Features      
Total Shows Overview
Movies vs TV Shows split
Country-wise analysis
Year-wise trends
Audience segmentation
Genre insights
